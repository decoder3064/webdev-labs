## Running the Labs Locally

Each lab is a React application built with Vite:

```bash
# Navigate to specific lab
cd lab-[number]

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173` (Vite's default port)
